## Getting Started

### Installation

1. Clone the repository
   ```
   git clone https://github.com/kbagdanskas/vetbeeBackEnd.git
   ```
2. Install dependencies
   ```
   npm install
   ```
3. And make sure to run it locally
   ```
   npm run dev
   ```
